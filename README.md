CapiBobba
- Descripción del proyecto

CapiBobba es un sitio web desarrollado como proyecto práctico, cuyo objetivo es presentar una tienda de bebidas tipo bubble tea.
El sitio muestra información sobre los productos (capibobbas), sucursales, horarios de atención y medios de pago, utilizando un diseño moderno y responsivo.

El proyecto sirve como práctica de maquetación web, uso de Bootstrap e interactividad básica con JavaScript y jQuery.

- Tecnologías utilizadas

HTML5
Estructura del sitio web y contenido principal.

CSS3
Estilos personalizados, colores, tipografías y efectos visuales (como blur y overlays).

Bootstrap 5
Diseño responsivo, grillas, navbar, cards, carrusel, modales y utilidades visuales.

JavaScript
Lógica básica del sitio e interacción con elementos.

jQuery
Manejo de eventos (click, doble click, hover) y manipulación del DOM.

Google Fonts
Tipografía utilizada en el proyecto.
